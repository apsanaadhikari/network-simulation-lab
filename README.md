# network-simulation-lab
Tool: Cisco Packet Tracer

## Description
A series of network simulation labs designed and configured 
using Cisco Packet Tracer to replicate real-world networking 
environments.

## Labs Covered
- VLAN configuration and segmentation
- DHCP and DNS setup
- NAT and PAT implementation
- Network troubleshooting and connectivity verification
